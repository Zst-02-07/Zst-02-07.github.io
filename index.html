LeetCode 1114.按序打印

发表于 2022-02-13

算法：
通过Object类的wait方法让线程等待，直到其他线程调用此对象的notifyAll方法，唤醒该对象等待的所有线程。
Object是三个线程的共享对象，通过对象锁机制可以控制线程执行顺序。
解题代码：
class Foo {
private Object object=new Object(); //对象锁
private int flag=1; //判断执行顺序
public Foo() {

}

public void first(Runnable printFirst) throws InterruptedException {
    synchronized(object){                            
        while(flag!=1) object.wait();
    // printFirst.run() outputs "first". Do not change or remove this line.
    printFirst.run();
    flag=2;
    object.notifyAll();
    }
}

public void second(Runnable printSecond) throws InterruptedException {
    synchronized(object){
        while(flag!=2) object.wait();
    // printSecond.run() outputs "second". Do not change or remove this line.
    printSecond.run();
    flag=3;
    object.notifyAll();
    }
}

public void third(Runnable printThird) throws InterruptedException {
     synchronized(object){
        while(flag!=3) object.wait();
    // printThird.run() outputs "third". Do not change or remove this line.
    printThird.run();
}
}

}

LeetCode 48.旋转图像

发表于 2022-02-13

算法(找规律)：
观察发现，顺时针旋转90°即可
顺时针旋转90°又可拆分为水平线翻折，再按主对角线翻折

解题代码：
class Solution {
public void rotate(int[][] matrix) {
for(int i=0;i<matrix.length/2;i++) //水平翻折
for(int j=0;j<matrix.length;j++){
int temp=matrix[i][j];
matrix[i][j]=matrix[matrix.length-1-i][j];
matrix[matrix.length-1-i][j]=temp;
}
for(int i=0;i<matrix.length;i++) //主对角线翻折
for(int j=0;j<i;j++){
int temp=matrix[i][j];
matrix[i][j]=matrix[j][i];
matrix[j][i]=temp;
}
}
}

LeetCode 55.跳跃游戏

发表于 2022-02-13

算法：能到达某一个位置，则在该位置之前的每个位置都能到达。
通过计算最远距离max之前每个位置能到达的最远距离d（下标+数组元素），从而不断更新最远距离max
如果最远距离max大于数组最后位置，返回true，其余返回else

解题代码：
class Solution {
public boolean canJump(int[] nums) {
int d=0,max=0;
for(int i=0;i<=max;i++){
d=i+nums[i];
max=Math.max(d,max);
if(max>=nums.length-1)
return true;
}
return false;
}
}

LeetCode 11.盛水最多的容器

发表于 2022-02-13

算法(双指针)：
1.规定两个指针i,j分别位于两端
2.在两指针相遇之前,计算面积s,更新面积的最大值max
3.选定短的，向中间移动，直至相遇

解题代码：
class Solution {
public int maxArea(int[] height) {
int i=0,j=height.length-1,s=0,max=0; //i从左开始移动,j从右开始移动,s代表面积,max代表最大面积
while(i<j){
s=(j-i)*Math.min(height[i],height[j]); //求面积
max=Math.max(s,max); //找出面积最大值
if(height[i]<height[j]){ //左右指针的移动
i++;
}
else j–;
}
return max;
}
}

LeetCode 292.Nim游戏

发表于 2022-02-13

算法(归纳推理)：
1.n为1，2，3时，自己获胜
2.n为4，对方获胜
3.n为5，6，7时，自己获胜
4.n为8，对方获胜
推理可得，n为4，8，……即4的倍数时，对方获胜。

解题代码：
class Solution {
public boolean canWinNim(int n) {
return n%4!=0;
}
}
